# 文法 
全部大文字にしたものは, 適宜別のキーワードにして使えるものです.
例: @COLOR_TEXT{text} は @red{text}や@lime{text}などのように使えます.

## 通常のマークダウン
markedを呼び出しているため, 部分的にmarkedの記法が使えます.

+ **太字** や _イタリック_
    + \*\*text\*, \_text\_
+ `インラインのソース`
    + \`text\`
+ > 引用
    + \> text

## 他では何の役にも立たないマークダウン 
\#記号から始まる命令です.
その行はタイトル行として処理されます.
先頭にスペースがあるとうまく動きません.

ボックスを閉じるには `!\# オプション1 | オプション2 | ...` という形式の閉じ命令が必要です.

ただし, 次の命令がボックス命令の場合は自動で閉じられます.


### 説明 Explain Box
説明ボックスは `#ex タイトル` の形式です.
水色の見出しが作られます.

閉じ命令のオプションは受け取りません.

### 例題 Example(exだとダブるので e.g.) Box
例題ボックスは `#eg タイトル` の形式です.
紫色の見出しが作られます.

閉じ命令のオプションは受け取りません.

### 練習 Practice Box
練習ボックスは `#pr タイトル` の形式です.
黄色の見出しが作られます.

閉じ命令のオプションは受け取りません.

### 課題 Assignment Box
課題ボックスは `#as タイトル` の形式です.
赤色の見出しが作られます.

閉じ命令のオプションは受け取りません.

**上の4つは色が違うだけで全く同じものです.**

### 問題 Problem Box
問題Boxは　`#pb title` の形式です.
緑色の見出しが作られます.

閉じ命令のオプションに, 答えのリストを受け取ります.

### 考察 Essay Box
考察Boxは　`#es title` の形式です.
黄緑色の見出しが作られます.

閉じ命令のオプション1のみに, はじめに表示する行数を受け取ります. がなんかうまく動いていません.

### ソースコード Code Box
コードBoxは　`#cd title` の形式です.
ピンク色の見出しが作られます.

閉じ命令のオプション1には ファイル名(main.cなど), オプション2には言語名(Cなど)を受け取りますが, ハイライトはされません. 


## インライン命令
インライン命令は @ 記号から始まり, 引数を波括弧{}で挟みます.
例: @CMD{arg1}{arg2}

### 色変更
形式:`@COLOR_NAME{text}`

まあまあな数の色名に対応していますが, カラーコードは未対応です.

### 画像
形式: `@img{src}{alt}{width}{height}`

srcのパスは相対パスがおすすめです.
widthとheightには \* (いいかんじにする)を指定できます.

## システム命令
システム命令とは(名ばかりですが)設定を書き換える命令です.

### Def命令
形式: `\def{@CMD_NAME}[NUM]{SOME_TAGS_OR_COMMANDS}`

インラインコマンドを自作出来ます.
引数は$1～$9まで指定可能です.
バグがあったら教えてください.

例: `\def{@mycolor}[1]{<span style="color:#123456">$1</span>}`

### タイトル命令 (重要)
形式: `\title{TITLE_TEXT}`

生成されたHTMLのタイトルを指定します.
また, データをローカル保存するための同一性確認にもこのタイトルを使います.

### ソース命令
形式: `\src{ROOT_PATH}`

画像などの読み込み時に, 先頭につけるパスを指定できます.

例: <br>
```tex
\src{.some/path/}
@img{image.png}{...}
```
とした場合は
```html
<img src=".some/path/image.png" ... >
```
のようになります.
